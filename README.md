# WaSHIVineyardLTARE_Baseline
WaSHIVineyardLTARE_Baseline contains data, code, and documentation supporting a baseline assessment of soil health indicators across multiple depths in a newly established, semi-arid vineyard within the Washington Soil Health Initiative LTARE network. It provides a reference for evaluating long-term soil health change in perennial vineyard systems.
